### ASP.NET Core でビューのレンダリング時にマルチバイト文字を HTML エンコードさせない

Startup.cs の ConfigureServices() 内で AddWebEncoders() のオプションで **TextEncoderSettings** を **UnicodeRanges.All** に設定します。
